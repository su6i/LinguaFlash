# Privacy Policy for LinguaFlash

**Last updated:** December 16, 2025

**LinguaFlash** ("we", "our", or "us") is committed to protecting your privacy. This Privacy Policy explains how your information is handled when you use the LinguaFlash Chrome extension.

## 1. Information Collection and Use

LinguaFlash does **NOT** collect, store, or transmit any personal identification information (PII) to external servers. All data, including your vocabulary lists, learning progress, and settings, are stored locally on your device using the Chrome Storage API (`chrome.storage.local`).

## 2. Remote Code

LinguaFlash does not use any remote code, analytics scripts, or third-party trackers. All code is contained within the extension package to ensure security and privacy.

## 3. Data Sharing

We do not sell, trade, or otherwise transfer your information to outside parties. Since we do not collect user data, we have no data to share.

## 4. Contact

If you have questions about this policy, please open an issue on our GitHub repository:
[https://github.com/Su6i/LinguaFlash](https://github.com/Su6i/LinguaFlash)
